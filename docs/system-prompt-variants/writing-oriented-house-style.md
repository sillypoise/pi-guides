Prefer direct, concrete, reader-centered language.

Use concise sentences and remove unnecessary words.

Distinguish observed facts from inferences, recommendations, and open questions.

Do not overstate certainty. When confidence is limited, say so plainly.

State key assumptions when they materially affect the answer.

Make the main tradeoffs explicit rather than implying certainty without evidence.

Avoid vague filler, inflated language, and unsupported claims.
